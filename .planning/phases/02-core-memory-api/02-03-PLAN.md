**app/routers/memory.py** -- Memory write endpoint:

Create `router = APIRouter(prefix="/v1", tags=["memory"])`.

Import the limiter from app.main: `from app.main import limiter`

Define background task helpers (module-level async functions):

`async def _generate_embedding(tenant_id: str, memory_id: str, text: str, app_state)`:
- Try to get embedding from `app_state.embedding_client.embed_single(text)`
- If embedding is None (breaker tripped or no API key), log warning and return
- Store embedding BLOB in tenant DB: `UPDATE memories SET embedding = ? WHERE id = ?` (serialize numpy array with `embedding.tobytes()`)
- Add to vector index: `app_state.index_manager.add_vector(tenant_id, memory_id, embedding)`
- Run cosine dedup check (post-embedding): query other memory embeddings from DB, use `check_cosine_duplicate`. If near-dupe found, soft-delete the new memory: `UPDATE memories SET is_deleted = 1 WHERE id = ?`
- Wrap entire body in try/except with logging (BackgroundTasks swallows exceptions silently -- Pitfall 6 from research)

`async def _extract_entities(tenant_id: str, memory_id: str, text: str, app_state)`:
- Get tenant conn from `app_state.tenant_manager.get_connection(tenant_id)`
- Call `process_entities_for_memory(conn, memory_id, text)` from entity_extraction service
- Wrap in try/except with logging

POST /v1/memory route -- apply rate limit decorator (MEM-12):
```python
@router.post("/memory", status_code=201)
@limiter.limit("60/minute")
async def create_memory(
    body: MemoryCreate,
    request: Request,
    background_tasks: BackgroundTasks,
    tenant: dict = Depends(get_tenant),
):
```

Implementation:
1. Get tenant connection: `conn = await request.app.state.tenant_manager.get_connection(tenant["id"])`
2. Compute text_hash: `th = text_hash(body.text)`
3. Check exact duplicate: query `SELECT id, text, user_id, agent_id, session_id, metadata, created_at, updated_at FROM memories WHERE text_hash = ? AND is_deleted = 0` using `th`
4. If duplicate found: log usage with `await log_usage(request.app.state.system_db, tenant["id"], "memory.create.duplicate", "/v1/memory", 200)`, return MemoryResponse with status="duplicate" and 200 status code (not 201)
5. If not duplicate:
   - Generate `memory_id = str(uuid.uuid4())`
   - `now = int(time.time())`
   - INSERT into memories: `(id, text, text_hash, user_id, agent_id, session_id, metadata, created_at, updated_at)` -- metadata serialized as `json.dumps(body.metadata or {})`
   - `await conn.commit()`
   - Queue background embedding: `background_tasks.add_task(_generate_embedding, tenant["id"], memory_id, body.text, request.app.state)`
   - Queue background entity extraction: `background_tasks.add_task(_extract_entities, tenant["id"], memory_id, body.text, request.app.state)`
   - Log usage: `await log_usage(request.app.state.system_db, tenant["id"], "memory.create", "/v1/memory", 201)`
   - Return MemoryResponse with status="created", HTTP 201
