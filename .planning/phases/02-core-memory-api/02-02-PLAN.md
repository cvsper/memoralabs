**app/services/embedding.py** -- Async Fireworks.ai client with circuit breaker:

```
class EmbeddingClient:
    def __init__(self, api_key: str, model: str, dim: int = 1024):
```

Methods:
- `__init__(self, api_key, model, dim=1024)`: store config, create `httpx.AsyncClient(timeout=30.0)`, init circuit breaker state: `_last_failure: float = 0.0`, `_cooldown: int = 120`, `_consecutive_failures: int = 0`, `_max_failures: int = 3`
- `_available(self) -> bool`: return False if no api_key. If `_consecutive_failures >= _max_failures`, check if cooldown elapsed since `_last_failure`. If elapsed, reset failures and return True. If not elapsed, return False.
- `async def embed(self, texts: list[str], batch_size: int = 20) -> np.ndarray | None`: if not available, return None. Loop over texts in batches of `batch_size`. For each batch, POST to `https://api.fireworks.ai/inference/v1/embeddings` with `{"model": self.model, "input": batch}` and `Authorization: Bearer {api_key}`. On 429: set `_last_failure = time.time()`, increment `_consecutive_failures`, return None. On success: extract embeddings from `resp.json()["data"]`, append. On connect/timeout error: trip breaker, return None. Return `np.array(all_embeddings, dtype=np.float32)`.
- `async def embed_single(self, text: str) -> np.ndarray | None`: convenience wrapper, calls `embed([text])`, returns first row or None.
- `async def close(self)`: close the httpx client.
- `@property def is_available(self) -> bool`: public accessor for `_available()`.

**app/services/vector_index.py** -- Per-tenant hnswlib index manager:

```
class TenantIndexManager:
    def __init__(self, data_dir: Path, dim: int = 1024, max_cached: int = 20):
```

Internal state:
- `_indexes: OrderedDict[str, hnswlib.Index]` -- LRU cache of loaded indexes
- `_id_maps: dict[str, list[str]]` -- maps integer position -> memory UUID per tenant
- `data_dir: Path` -- directory for index files (`{data_dir}/indexes/`)
- `dim: int` -- embedding dimension (1024)
- `max_cached: int` -- max indexes in memory
- `_lock: asyncio.Lock` -- guards all mutations
- `INITIAL_MAX_ELEMENTS = 10000`
- `GROWTH_FACTOR = 2`
- `RESIZE_THRESHOLD = 0.8` (resize when 80% full)

Methods:
- `_index_path(self, tenant_id: str) -> Path`: returns `self.data_dir / f"{tenant_id}.idx"`
- `_id_map_path(self, tenant_id: str) -> Path`: returns `self.data_dir / f"{tenant_id}.ids.json"`
- `_save_index(self, tenant_id, index, id_map)`: save index file + id_map JSON to disk. Run in executor to avoid blocking event loop (hnswlib save is synchronous C++ I/O).
- `_load_or_create(self, tenant_id) -> tuple[hnswlib.Index, list[str]]`: if index file exists, load it + load id_map JSON. Otherwise create new index with `space="cosine"`, `dim=self.dim`, `init_index(max_elements=INITIAL_MAX_ELEMENTS, ef_construction=200, M=16)`, `set_ef(200)`, empty id_map.
- `async def get_index(self, tenant_id) -> tuple[hnswlib.Index, list[str]]`: acquire lock. If cached, move_to_end and return. If at capacity, evict LRU (save then remove). Load or create. Cache and return.
- `async def add_vector(self, tenant_id, memory_id: str, embedding: np.ndarray)`: get index + id_map. Check if approaching capacity (`len(id_map) >= max_elements * RESIZE_THRESHOLD`), resize if needed. Append memory_id to id_map, get position (len - 1). Call `index.add_items(embedding.reshape(1, -1), np.array([position]))`. Save index to disk.
- `async def search(self, tenant_id, query_embedding: np.ndarray, k: int = 10, candidate_ids: set[str] | None = None) -> list[tuple[str, float]]`: get index. If index has 0 elements, return []. Call `knn_query(query_embedding.reshape(1, -1), k=min(k * 3, index.get_current_count()))` to over-fetch for filtering. Convert positions to memory_ids via id_map. Compute similarity = 1.0 - distance. If candidate_ids provided, filter results to only those IDs. Return top-k `[(memory_id, similarity_score)]` sorted by score desc.
- `async def remove_vector(self, tenant_id, memory_id: str)`: get index + id_map. Find position of memory_id in id_map. Call `index.mark_deleted(position)`. Note: do NOT remove from id_map (hnswlib positions are permanent). Save index.
- `async def save_all(self)`: iterate all cached indexes and save to disk. Called on shutdown.
- `async def close(self)`: save_all then clear caches.

**Rate limiting setup (MEM-12)** -- Wire per-tenant rate limiting BEFORE lifespan:

Define tenant-aware rate limit key function at module level:
```python
def get_tenant_key(request: Request) -> str:
    """Rate limit by tenant API key hash (or IP as fallback)."""
    auth = request.headers.get("Authorization", "")
    if auth.startswith("Bearer "):
        return hashlib.sha256(auth[7:].encode()).hexdigest()[:16]
    return get_remote_address(request)
```

Instantiate limiter at module level (so routers can import it):
```python
limiter = Limiter(key_func=get_tenant_key)
```

After creating the FastAPI app instance:
```python
app.state.limiter = limiter
app.add_exception_handler(RateLimitExceeded, _rate_limit_exceeded_handler)
```

This exports `limiter` from `app.main` so that routers in plans 02-03 and 02-05 can import it and apply `@limiter.limit()` decorators to individual routes:
