**GET /v1/memory** -- List memories with pagination:
```python
@router.get("/memory")
async def list_memories(
    request: Request,
    page: int = Query(1, ge=1),
    page_size: int = Query(20, ge=1, le=100),
    user_id: str | None = Query(None),
    agent_id: str | None = Query(None),
    session_id: str | None = Query(None),
    tenant: dict = Depends(get_tenant),
):
```
- Get tenant conn
- Build WHERE clause: `is_deleted = 0` always. Add `user_id = ?`, `agent_id = ?`, `session_id = ?` if provided.
- Count total: `SELECT COUNT(*) FROM memories WHERE {conditions}`
- Fetch page: `SELECT id, text, user_id, agent_id, session_id, metadata, created_at, updated_at FROM memories WHERE {conditions} ORDER BY created_at DESC LIMIT ? OFFSET ?` with `page_size` and `(page - 1) * page_size`
- Parse metadata JSON for each row
- Log usage: `await log_usage(request.app.state.system_db, tenant["id"], "memory.list", "/v1/memory", 200)`
- Return MemoryListResponse with memories, total, page, page_size

**GET /v1/memory/{memory_id}** -- Get single memory:
```python
@router.get("/memory/{memory_id}")
async def get_memory(
    memory_id: str,
    request: Request,
    tenant: dict = Depends(get_tenant),
):
```
- Get tenant conn
- Query: `SELECT * FROM memories WHERE id = ? AND is_deleted = 0`
- If not found: raise HTTPException(404, detail="Memory not found")
- Update access_count and last_accessed: `UPDATE memories SET access_count = access_count + 1, last_accessed = ? WHERE id = ?`
- Commit
- Log usage: `await log_usage(request.app.state.system_db, tenant["id"], "memory.get", f"/v1/memory/{memory_id}", 200)`
- Return MemoryResponse (status="created" for existing memories -- the status field reflects creation state)

**GET /v1/memory/{memory_id}/entities** -- Get entities for a memory (RETR-03):
```python
@router.get("/memory/{memory_id}/entities")
async def get_memory_entities(
    memory_id: str,
    request: Request,
    tenant: dict = Depends(get_tenant),
):
```
- Get tenant conn
- Verify memory exists: `SELECT id FROM memories WHERE id = ? AND is_deleted = 0`
- If not found: raise HTTPException(404, detail="Memory not found")
- Query entities linked to this memory:
  ```sql
  SELECT e.id, e.name, e.entity_type, e.name_normalized, e.created_at
  FROM entities e
  JOIN memory_entities me ON e.id = me.entity_id
  WHERE me.memory_id = ?
  ```
- Query relations linked to this memory:
  ```sql
  SELECT r.id, r.source_entity_id, r.relationship, r.target_entity_id,
         se.name as source_name, se.entity_type as source_type,
         te.name as target_name, te.entity_type as target_type
  FROM relations r
  JOIN entities se ON r.source_entity_id = se.id
  JOIN entities te ON r.target_entity_id = te.id
  WHERE r.memory_id = ?
  ```
- Log usage: `await log_usage(request.app.state.system_db, tenant["id"], "memory.entities", f"/v1/memory/{memory_id}/entities", 200)`
- Return `{"memory_id": memory_id, "entities": [...], "relations": [...], "total_entities": len(entities), "total_relations": len(relations)}`

IMPORTANT: Define the `/memory/{memory_id}/entities` route BEFORE any other `/{memory_id}` sub-path routes to avoid FastAPI path matching conflicts.

**PATCH /v1/memory/{memory_id}** -- Update memory:
```python
@router.patch("/memory/{memory_id}")
async def update_memory(
    memory_id: str,
    body: MemoryUpdate,
    request: Request,
    background_tasks: BackgroundTasks,
    tenant: dict = Depends(get_tenant),
):
```
- Get tenant conn
- Verify memory exists: `SELECT id FROM memories WHERE id = ? AND is_deleted = 0`
- If not found: raise HTTPException(404)
- Build SET clause dynamically from non-None fields in body. If body.text is set: also update text_hash, and queue re-embedding in background. Always set `updated_at = int(time.time())`.
- If body.metadata is set: serialize as JSON
- Execute UPDATE with dynamic SET clause
- Commit
- If text changed: queue background re-embedding (`_generate_embedding` from the existing module-level function) and re-entity-extraction
- Log usage: `await log_usage(request.app.state.system_db, tenant["id"], "memory.update", f"/v1/memory/{memory_id}", 200)`
- Fetch and return updated memory as MemoryResponse

**DELETE /v1/memory/{memory_id}** -- Soft-delete memory:
```python
@router.delete("/memory/{memory_id}")
async def delete_memory(
    memory_id: str,
    request: Request,
    tenant: dict = Depends(get_tenant),
):
```
- Get tenant conn
- Verify memory exists: `SELECT id FROM memories WHERE id = ? AND is_deleted = 0`
- If not found: raise HTTPException(404)
- Soft-delete: `UPDATE memories SET is_deleted = 1, updated_at = ? WHERE id = ?`
- Commit
- Remove from vector index if available: `request.app.state.index_manager.remove_vector(tenant["id"], memory_id)` (try/except, non-fatal)
- Log usage: `await log_usage(request.app.state.system_db, tenant["id"], "memory.delete", f"/v1/memory/{memory_id}", 200)`
- Return `{"id": memory_id, "status": "deleted"}`
