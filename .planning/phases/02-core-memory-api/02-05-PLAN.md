**app/services/search.py** -- Search orchestrator:

```python
async def search_memories(
    conn: aiosqlite.Connection,
    embedding_client: EmbeddingClient,
    index_manager: TenantIndexManager,
    tenant_id: str,
    query: str,
    user_id: str | None = None,
    agent_id: str | None = None,
    session_id: str | None = None,
    metadata_filter: dict | None = None,
    metadata_filter_operator: str = "and",
    limit: int = 10,
) -> list[dict]:
```

Implementation steps (metadata-first pattern per RETR-05 and research Pattern 4):

1. **Build SQL filter for candidate set**: Start with `WHERE is_deleted = 0 AND embedding IS NOT NULL` (only search memories that have embeddings). Add `AND user_id = ?` if user_id provided. Add `AND agent_id = ?` if agent_id provided. Add `AND session_id = ?` if session_id provided.

   For metadata_filter (MEM-03 AND/OR support):
   - If `metadata_filter_operator == "and"` (default): iterate key-value pairs, add `AND json_extract(metadata, '$.{key}') = ?` for each pair. All conditions must match.
   - If `metadata_filter_operator == "or"`: iterate key-value pairs, build a grouped OR clause: `AND (json_extract(metadata, '$.{key1}') = ? OR json_extract(metadata, '$.{key2}') = ? OR ...)`. At least one condition must match.

2. **Fetch candidate IDs + created_at**: `SELECT id, text, user_id, agent_id, session_id, metadata, created_at FROM memories WHERE {conditions}`. Store as a dict `{id: row_data}`.

3. **If no candidates, return empty list.**

4. **Embed the query**: `query_embedding = await embedding_client.embed_single(query)`. If None (breaker tripped), fall back to returning candidates sorted by created_at DESC (recency fallback -- better than empty results).

5. **Vector search with candidate filter**: `results = await index_manager.search(tenant_id, query_embedding, k=limit, candidate_ids=set(candidates.keys()))`. This returns `[(memory_id, raw_score)]`.

6. **Apply temporal decay**: For each result, look up `created_at` from candidates dict. `final_score = apply_decay(raw_score, created_at)`.

7. **Sort by final_score DESC, take top `limit`.**

8. **Build result dicts**: For each result, merge memory data from candidates dict with score. Parse metadata JSON. Return list of dicts matching MemorySearchResult fields.

Also add a helper:

```python
async def count_tenant_memories(conn: aiosqlite.Connection) -> int:
    """Count non-deleted memories for DX-05 memories_used."""
```
Returns `SELECT COUNT(*) FROM memories WHERE is_deleted = 0`.
