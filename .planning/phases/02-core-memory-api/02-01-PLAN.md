Create `app/services/` directory with `__init__.py`.

**app/services/dedup.py** -- Port from ZimMemory `text_hash()` and add cosine dedup helper:
- `text_hash(text: str) -> str`: `hashlib.sha256(text.strip().lower().encode()).hexdigest()[:32]` -- deterministic hash for exact dedup
- `async def check_exact_duplicate(conn, text: str) -> str | None`: query `SELECT id FROM memories WHERE text_hash = ? AND is_deleted = 0`, return existing memory_id or None
- `def check_cosine_duplicate(embedding: np.ndarray, candidate_embeddings: list[tuple[str, np.ndarray]], threshold: float = 0.95) -> str | None`: compute cosine similarity against candidates, return memory_id of first match above threshold, or None. This is for background post-embedding dedup check.

**app/services/decay.py** -- Port from ZimMemory `apply_decay()`:
- `DECAY_HALF_LIFE_DAYS = 30`
- `def apply_decay(score: float, created_at: int) -> float`: blend 80% base score + 20% recency bonus where recency = `0.5 ** (age_days / DECAY_HALF_LIFE_DAYS)`. Uses `time.time()` for current time.
- `def decay_factor(created_at: int) -> float`: returns raw decay factor (0.0-1.0) for a timestamp. Useful for batch scoring.

**app/services/entity_extraction.py** -- Port ZimMemory regex entity/relation extraction:
- `def normalize_entity_name(name: str) -> str`: `re.sub(r'[^a-z0-9\s]', '', name.lower()).strip()`
- Entity patterns (at module level): detect person names (2-3 capitalized words), organizations (Inc, Corp, LLC, Ltd, Co), locations (known city/country patterns or "in [Capitalized]"), dates (ISO dates, "January 2024", "last Tuesday", etc), topics (after "about", "regarding")
- `def extract_entities(text: str) -> list[dict]`: returns `[{"name": str, "type": "person"|"organization"|"location"|"date"|"topic", "span": (start, end)}]`. Use at least 10 regex patterns ported from ZimMemory.
- `def extract_relations(text: str, entities: list[dict]) -> list[dict]`: returns `[{"source": str, "relationship": str, "target": str, "source_type": str, "target_type": str}]`. Patterns: "X works at Y", "X met Y", "X lives in Y", "X is Y", "X founded Y", "X manages Y", "X reports to Y".
- `async def find_or_create_entity(conn, name: str, entity_type: str) -> str`: check if entity exists by name_normalized, create if not, return entity_id. Uses uuid4 for new IDs.
- `async def process_entities_for_memory(conn, memory_id: str, text: str) -> dict`: orchestrator -- extracts entities and relations, persists to entities/relations tables, returns `{"entities_found": int, "relations_found": int}`.

**app/deps.py** -- FastAPI dependency injection:
- `async def get_tenant(request: Request) -> dict`: extract `Authorization: Bearer <key>` header, SHA-256 hash it, call `get_tenant_by_key_hash(app.state.system_db, key_hash)`. Raise HTTPException(401) if missing/invalid. Return tenant dict.
- `async def get_tenant_conn(request: Request, tenant: dict = Depends(get_tenant)) -> aiosqlite.Connection`: get connection from `request.app.state.tenant_manager.get_connection(tenant["id"])`. Return the connection.

**app/models/memory.py** -- Pydantic v2 models for memory endpoints:
- `MemoryCreate(BaseModel)`: text (str, required, min 1 char), user_id (str | None), agent_id (str | None), session_id (str | None), metadata (dict | None = None)
- `MemoryResponse(BaseModel)`: id, text, user_id, agent_id, session_id, metadata, created_at, updated_at, status (Literal["created", "duplicate"])
- `MemoryUpdate(BaseModel)`: text (str | None = None), metadata (dict | None = None), user_id (str | None = None), agent_id (str | None = None), session_id (str | None = None) -- all optional
- `MemorySearchRequest(BaseModel)`: query (str, required), user_id (str | None), agent_id (str | None), session_id (str | None), metadata_filter (dict | None = None), limit (int = 10, ge=1, le=100)
- `MemorySearchResult(BaseModel)`: id, text, score (float), metadata, user_id, agent_id, session_id, created_at
- `MemorySearchResponse(BaseModel)`: results (list[MemorySearchResult]), total (int), memories_used (int), memories_limit (int)
- `MemoryListResponse(BaseModel)`: memories (list[MemoryResponse]), total (int), page (int), page_size (int)
