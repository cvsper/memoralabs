  <name>Task 1: TenantDBManager implementation</name>
  <files>

  <action>
    Create `app/db/manager.py` with `TenantDBManager` class. Follow the RESEARCH.md pattern closely but add path validation and schema init.

    **Constructor** `__init__(self, data_dir: Path, max_connections: int = 50)`:
    - Store data_dir (Path), max_connections (int)
    - Create `_pool: OrderedDict[str, aiosqlite.Connection]` (empty)
    - Create `_lock: asyncio.Lock()`
    - Create tenant subdirectory: `(data_dir / "tenants").mkdir(parents=True, exist_ok=True)`

    **Private** `_validate_tenant_id(self, tenant_id: str) -> None`:
    - Validate tenant_id matches UUID pattern: `^[a-f0-9]{8}-[a-f0-9]{4}-[a-f0-9]{4}-[a-f0-9]{4}-[a-f0-9]{12}$` (lowercase hex + dashes)
    - Raise `ValueError("Invalid tenant_id format")` if not matching
    - This prevents path traversal attacks (../../etc)

    **Private** `_tenant_db_path(self, tenant_id: str) -> Path`:
    - Call `_validate_tenant_id(tenant_id)`
    - Return `self.data_dir / "tenants" / f"{tenant_id}.db"`

    **Async** `get_connection(self, tenant_id: str) -> aiosqlite.Connection`:
    - Acquire `_lock`
    - If tenant_id in pool: `move_to_end(tenant_id)`, return cached connection
    - If pool at capacity: `popitem(last=False)` to get LRU entry, `await evicted_conn.close()`
    - Open new connection: `aiosqlite.connect(str(self._tenant_db_path(tenant_id)))`
    - Set 4 PRAGMAs: journal_mode=WAL, synchronous=NORMAL, cache_size=2000, foreign_keys=ON
    - Set `conn.row_factory = aiosqlite.Row`
    - Store in pool, return

    **Async** `create_tenant_db(self, tenant_id: str) -> Path`:
    - Get db_path via `_tenant_db_path(tenant_id)`
    - If db_path.exists(): raise `ValueError(f"Tenant DB already exists: {tenant_id}")`
    - Get connection via `get_connection(tenant_id)` (creates the file)
    - Call `init_tenant_db(conn)` from `app.db.tenant`
    - Return db_path

    **Async** `close_connection(self, tenant_id: str) -> None`:
    - Acquire lock, if tenant_id in pool: close connection, delete from pool

    **Async** `close_all(self) -> None`:
    - Acquire lock, close all connections, clear pool

    **Property** `pool_size(self) -> int`:
    - Return `len(self._pool)`

    Import `asyncio`, `aiosqlite`, `re`, `OrderedDict` from collections, `Path` from pathlib, `init_tenant_db` from `app.db.tenant`.
  </action>
