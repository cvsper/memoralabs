<task type="auto">
  <name>Task 2: Mount static files and add landing page route</name>
  <files>app/main.py</files>
  <action>
In `app/main.py`, add the following:

1. Add imports at top:
```python
from pathlib import Path
from fastapi.responses import HTMLResponse
from fastapi.staticfiles import StaticFiles
```

2. Add a route for `GET /` that reads and serves the landing page HTML. Place this AFTER the exception handlers and BEFORE `app.include_router(...)` calls:

```python
_LANDING_HTML = (Path(__file__).parent / "static" / "landing.html").read_text()

@app.get("/", response_class=HTMLResponse, include_in_schema=False)
async def landing_page():
    """Serve the landing page."""
    return _LANDING_HTML
```

Cache the HTML at module level via `_LANDING_HTML` so the file is read once at import time, not on every request.

3. Mount static files AFTER all `app.include_router()` calls (mount order matters in FastAPI — later mounts do not shadow earlier routes):

```python
app.mount("/static", StaticFiles(directory=str(Path(__file__).parent / "static")), name="static")
```

IMPORTANT: The `/static` mount must come AFTER all router includes and route definitions. Do NOT mount at `/` — that would shadow all API routes.

4. Verify the `include_in_schema=False` is set on the landing route so it does not appear in `/docs`.
